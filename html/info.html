The architecture of Maltese megalithic temples has evolved over time, as can be seen above. Indeed, most of these temples were built during the "temple phase" (4,100 - 2,500 B.C.) which is a period divided into several phases :
• 4,100 - 3,800 BC : phase Żebbuġ
• 3,800 - 3,600 BC : phase L-Imġarr
• 3,600 - 3,000 BC : phase Ġgantija
• 3,000 - 2,900 BC : phase Hal Saflieni
• 2,900 - 2,500 BC : phase Hal Tarxien
